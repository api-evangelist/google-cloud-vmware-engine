Feature VMware Engine introduces enhancements to simplify using external NFS datastores. External NFS datastores let you scale storage independently of compute resources for your VMware workloads. You can use Filestore or Google Cloud NetApp Volumes as external NFS datastores for ESXi hosts in Google Cloud VMware Engine. For more information, see NFS datastores overview .
