Feature Preview : You can configure disaster recovery for Google Cloud VMware Engine by using JetStream. Disaster recovery options include the following: Set up a Google Cloud VMware Engine private cloud as a recovery site for your on-premises VMware applications. Protect workloads on a primary private cloud by using a recovery site in another Google Cloud location or on-premises environment.
